# 📄 PDF Chatbot

A simple PDF Question Answering chatbot built using **PyMuPDF**, **Sentence Transformers**, **FAISS**, and **Microsoft Phi-3 Mini**.

## 🚀 Features

✅ Extract text from PDF files  
✅ Fixed-size chunking  
✅ Sentence-based chunking  
✅ Generate embeddings using Sentence Transformers  
✅ Semantic search with Cosine Similarity  
✅ Vector Search using FAISS  
✅ Metadata filtering (Page-wise retrieval)  
✅ Context-based Question Answering  
✅ Local LLM inference using Phi-3 Mini
